# LightManagerBot

## Installation

### Prerequisites :
* Visual Studio 2015
* An account on [Luis plateform](https://www.luis.ai/) to implement and register your training model


When these prerequisites are installed, to initialize the solution you must go to the root folder and open this file :
LightManagerBot.sln

## Development
This app interacts with [Bot Framework](https://dev.botframework.com/) in order to etablish a human interaction (dialog).
